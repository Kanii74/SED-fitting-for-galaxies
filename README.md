# Scientific Background

Galaxies emit radiation across a wide range of wavelengths, from the **ultraviolet (UV)** to the **far-infrared (FIR)**. The **Spectral Energy Distribution (SED)** represents the distribution of flux as a function of wavelength and provides valuable information about the physical properties of galaxies.

By modeling the SED of a galaxy, we can estimate key astrophysical parameters such as:

- Star Formation Rate (SFR)
- Stellar Mass
- Redshift
- Age of the Stellar Population
- Star Formation History

SED fitting involves comparing **observed photometric fluxes** with **synthetic spectra generated by theoretical models**. Using Bayesian inference, it is possible to determine the posterior probability distributions of these parameters.

The galaxies analyzed in this assignment belong to the **GAMA (Galaxy And Mass Assembly) Survey**, which is a large spectroscopic and imaging survey of approximately **300,000 galaxies covering about 290 square degrees of the sky**. :contentReference[oaicite:1]{index=1}

# Methodology

The modelling of galaxy spectra was carried out using the following steps.

## 1. Data Preparation

Photometric flux values and their associated uncertainties were extracted from the ASCII datasets and formatted as input for the **BAGPIPES** fitting routines.

In addition, **filter response curves** for the photometric bands were obtained from the **SVO Filter Profile Service**, which are required to accurately model photometric observations.

---

## 2. Installation of BAGPIPES

The **BAGPIPES** package was installed in a Python environment compatible with **Python 3.10.9**.

BAGPIPES enables:

- Generation of synthetic galaxy spectra
- Photometric and spectroscopic fitting
- Bayesian parameter estimation
- Star formation history reconstruction

---

## 3. Model Definition

A galaxy model was defined within BAGPIPES with parameters including:

- Star Formation History (SFH)
- Stellar Mass
- Redshift
- Age of Stellar Population
- Dust Attenuation
- Nebular Emission (when applicable)

These parameters define the **synthetic spectral energy distribution**, which is compared with observed photometric flux values.

---

## 4. Model Fitting

BAGPIPES generates a **synthetic SED** and compares it with observed photometric measurements.

Using **Bayesian sampling techniques**, the code produces **posterior probability distributions** for each parameter.

The fitting process produces:

- Best-fit synthetic SED spectra
- Star formation history plots
- Estimates of galaxy physical parameters

# Comparison: 5-Band vs 21-Band Photometry

The analysis demonstrates that **21-band photometry provides more reliable constraints** on galaxy parameters compared to 5-band photometry.

## Reasons

### 1. Increased Data Points

A larger number of photometric bands provides more observational constraints on the SED model, reducing parameter degeneracies.

### 2. Broader Wavelength Coverage

21-band photometry spans wavelengths from **ultraviolet to far-infrared**, capturing spectral features that are not accessible with the limited SDSS bands.

### 3. Improved Parameter Estimation

The wider spectral coverage improves the accuracy of estimates for:

- Redshift
- Dust attenuation
- Star formation history
